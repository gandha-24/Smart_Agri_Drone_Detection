# ESP32-CAM AI Animal Detection & Alert System

## 📌 Overview
This project is a smart surveillance and animal deterrent system using ESP32-CAM and Python-based AI. It detects humans and animals in real-time and triggers alerts.

## 🚀 Features
- 📷 Live video streaming using ESP32-CAM  
- 🤖 Real-time object detection (Human, Cat, Rat)  
- 🔊 Automatic sound alerts via speaker  
- 📲 Mobile notification using WiFi (Telegram)  
- ⚡ Low-cost IoT + AI integration  

## 🧠 How It Works
1. ESP32-CAM streams live video over WiFi  
2. Python AI model processes the video  
3. Object is detected (person / animal)  
4. Command sent to ESP32  
5. ESP32 plays alert sound and sends notification  

## 🛠️ Tech Stack
- ESP32-CAM (Embedded System)  
- Python (OpenCV, cvlib)  
- Arduino IDE  
- WiFi Communication  
- Telegram API  

## 🔧 Hardware Required
- ESP32-CAM  
- Speaker / Buzzer  
- (Optional) Amplifier Module  
- Power Supply  

## ▶️ Setup
1. Upload ESP32 code using Arduino IDE  
2. Connect ESP32-CAM to WiFi  
3. Run Python detection script  
4. Configure ESP32 IP in Python code  
5. Setup Telegram Bot for notifications  

## 📷 Output
- Detects object in real-time  
- Plays alert sound  
- Sends mobile notification
